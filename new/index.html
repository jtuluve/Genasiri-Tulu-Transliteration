To enable special charcters of tulu language use "V" after special character. Ex:
1. "ಮಲ್ಪುವೆV" returns "mlApuov" and "ಮಲ್ಪುವೇV" returns "mlApuOv"
2. "ಎV" returns "oA" and "ಏV" returns "OA"

Tulu Transliterator

Transform text from various Indian scripts to beautiful Tulu script

About this tool: This tool helps to 'transliterate' (changes to other script, not other language - do not confuse with translate) the given text to Tulu script. Select the language you want to transliterate from and enter the text in the input box. The tool instantly gives the output in the result box. You can copy the result, which will be in English (Baravu code) and apply one of the Tulu fonts available here in any editing software you like.
